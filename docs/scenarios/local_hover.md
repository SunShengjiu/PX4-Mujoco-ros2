# 本地可视化悬停

## 目的

当你只想立刻看到 MuJoCo 中的飞机运动，而不想等待 PX4、ROS 2 或 QGroundControl 时，使用这条路径。

## 命令

```bash
make run-local
```

## 会启动什么

- MuJoCo
- `bridge.py`
- bridge 本地起飞/悬停控制器

这条路径不会启动 PX4。

## 预期结果

- MuJoCo 窗口打开。
- 飞机从地面升起。
- 飞机稳定在配置的目标高度附近。

## 注意事项

- 这是仓库里最快的可视化验证路径。
- 适合确认 MuJoCo 场景和飞行动作器是否正常。
- 这条路径不能验证 PX4 消息链路。
